## Быстрые правила
1) **В `main` напрямую не пушим.** Любые изменения — только через **Pull Request (PR)**.
2) **Один файл = один билет/тема.** Это уменьшает конфликты.

## Нейминг файлов
Файлы называем так: `ticketsNN.tex`, где `NN` — номер (01, 02, 03…).
